2016年最佳高层建筑奖公布　上海武汉入围

文章出处：中国建筑新闻网 发表时间：2016-06-27

  世界高层建筑与都市人居学会(Council on Tall Building and Urban Habitat )公布了第15届CTBUH最佳高层建筑奖项。超过100座提名建筑中评选出了四个大区的最佳建筑，包括美洲、亚洲&大洋洲、欧洲以及中东&非洲四个部分，同时颁发的还有城市人居奖、创新奖、建筑效能奖以及10年特别奖。CTBUH 之后还将从四个区域的优胜者中评选出世界最佳高层建筑。

  获奖建筑由世界知名事务所组成的委员会选出，评选标准根据建筑的各方面表现，特别强调“对于使用建筑个体和其所在城市具有积极影响的建筑。”

   最佳高层建筑 - 美洲: VIA 57 West / BIG

　　“VIA 57 West 同时受到了传统街区庭院和高层建筑的启发，它复杂的体型和扭转的建筑形式让使用者具有了最好的朝向哈德逊河的景观视野，并且让纽约的滨水环境在整合城市用地发展上提升至了动态的新标准。” - 评委 Michael Palladino, 理查德·迈耶事务所设计合伙人。

　　提名奖: 432 Park Avenue / Rafael Viñoly 建筑事务所、PNC 广场塔楼 / Gensler、Torre Reforma / LBR&A

最佳高层建筑 - 亚太地区: 上海中心 / Gensler

　　“上海中心展示了高层摩天楼对于公共空间的极大热诚。它采用了全球范围内首个真正意义上利用在摩天楼上的双层皮幕墙，不仅具有很好的绿色作用，同时也改善了整座建筑的通风系统。上海中心牺牲了部分使用面积来提供公共空间，让它走在了单纯商业需求的前面。” - 评委 Antony Wood, CTBUH 执行总监

　　提名奖: 海滩大道 / 福斯特建筑事务所(Foster + Partners), 北京绿地大望京中心 / Skidmore, Owings & Merrill，江西南昌绿地中央广场 / Skidmore, Owings & Merrill, 新世界国际中心 / Olson Kundig, 新加坡天空住宅 / 萨夫迪建筑事务所(Safdie Architects)

最佳高层建筑 - 欧洲: 白墙大楼 / 让·努维尔建筑事务所(Ateliers Jean Nouvel)

　　“白墙大楼是对于建筑物质性一次掷地有声的实践，在垂直轴线上成为了整个地中海区域建筑和环境价值的一次成功表达。北立面上延伸的植被和南立面上的凉廊创造了与自然之间的真正联系，而大楼上穿插的混凝土墙壁让建筑与当地的绿植真正交融在了一起。” - 评委 Karl Fender, Fender Katsalidis 建筑事务所总监

　　提名奖: 伊斯坦布尔 Allianz 大楼 / FXFOWLE、米兰 Allianz 大楼 / Arata Isozaki + Andrea Maffei, ECB - 欧洲中央银行 / 蓝天组(Coop Himmelb(l)au)、Grattacielo Intesa Sanpaolo Torino / 伦佐·皮亚诺建筑事务所(Renzo Piano Building Workshop)

最佳高层建筑 - 中东&非洲: 方块大楼 / Orange Architects

　　“方块代表了全球范围内城市高层住宅的典型形态，这座建筑利用一系列堆叠的体量创造了复杂而独特的空中别墅。大楼的结构设计非常成功，具有优雅的钢结构墙壁系统，视觉上具有标志性，同时可以进行自由的平面布局。” - 评委 Hashimah Hashim, KLCC 房地产公司总监

　　提名奖: Iris 海湾 / Atkins

城市人居奖: 武汉天地A地块 / Skidmore, Owings & Merrill

　　“武汉天地综合开发项目展示了高层住区在进行总体规划时，激活城市空间、提供高水准的娱乐、步行和社交空间的可能。高层建筑的布局与具有活力的公共空间相结合，创造了新社区中罕见的生动场景。武汉天地综合体为其中居住、工作和参观的人提供了高质量的环境，可以与世界其他任何地方建立已久的社区相媲美。” 评委 James Parakh，多伦多城市规划局城市设计经理。

10年特别奖: 赫斯特大厦 / 福斯特建筑事务所(Foster + Partners)

　　“在赫斯特大厦旁行走，你可能不会意识到正走在2000年纽约最重要的建筑成就旁边。这座建筑直接建造在一座1920年代办公建筑的遗址之上，这座建筑展现了在高密度城区保存低层历史建筑的另类做法。其经典的保护建筑和上方现代的钻石般结构并置，形成了鲜明的对比。”CTBUH 贡献者 Timothy Johnson，NBBJ 设计合伙人。

建筑效能奖: 台北101大楼 / 李祖原建筑事务所(C.Y. Lee)

　　“很少能见到对于现有建筑的改造能够达到如此的能源标准。这座建筑在能源和可持续性上的升级掷地有声。台北101大楼在经过异常艰苦的努力之后成为了世界上最具可持续性的高层建筑之一，最后的结果非常成功。除了一系列绿色技术和新安装的系统，严格的用户参与系统也让这座建筑独树一帜。”技术顾问 Juror Bill Browning，Terrapin Bright Green 联合创始人

创新奖: Pin-Fuse 系统

　　“Pin-Fuse 开启了高强度地震区域建筑柔性系统的新纪元。通过在预定的轴线上施加一定的应力，系统在地震区建筑的维护频率、花费、结构寿命上都具有创新性的贡献。”技术评委组长 SawTeen See，Leslie E. Robertson 事务所管理合伙人。
